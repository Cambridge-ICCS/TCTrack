Tracking Algorithms
===================

TCTrack contains a number of different methods for generating and processing cyclone
tracks from climate data.
These documents provide an overview of of how to use the different functionalities of
the code.

For full details of the API for each of the components please see the 
:doc:`TCTrack API documentation <../api/index>`.
